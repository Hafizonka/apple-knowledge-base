# Apple Knowledge Base

Интерактивная база знаний по технике Apple для стажеров.

Открывается как обычная HTML-страница:

- локально: `index.html`
- через GitHub Pages после публикации: ссылка вида `https://<user>.github.io/<repo>/`

Внутри:

- iPhone 13-17 Pro Max
- iPad 11, iPad Air M4, iPad Pro M5
- MacBook Air/Pro
- Apple Watch
- AirPods и аксессуары
- поиск, светлая/темная тема, карточки цветов и отличия моделей
